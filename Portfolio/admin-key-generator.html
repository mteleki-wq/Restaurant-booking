Admin key generator

Set your own admin password

Type a new password below to generate its secure hash. This never leaves your device — everything happens right here on the page.

  1. Type your new password above and tap Generate hash.
  2. Tap Copy hash.
  3. Open tier-selection-section.html in a text editor (or the Files app "Open in" a code editor).
  4. Find the line starting with const ADMIN_KEY_HASH = near the top of the admin script.
  5. Replace the long text between the quotes with your copied hash, then save.
