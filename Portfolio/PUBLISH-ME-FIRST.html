You're one page away

Put your portfolio live on the web.

Your portfolio is a single file — that means you can host it yourself, for free, forever, with no monthly hosting fees and no lock-in. Here's the whole process, right from your iPad or any browser. No coding, no apps to install.

About 10 minutes, once

Create a free GitHub account

GitHub is a trusted platform (owned by Microsoft) that hosts websites for free. Go to github.com/signup and sign up with your email — just like any other account.

Tip: choose a username you're happy to have in your web address, e.g. jordanavery becomes jordanavery.github.io.

Create your website's home

Once signed in, tap the + at the top right and choose New repository. A repository is simply the folder that holds your website.

Name it exactly: yourusername.github.io — replacing yourusername with your actual GitHub username. This exact name is what switches on free hosting.

Leave everything else as it is, tick Add a README file, and tap Create repository.

Upload your portfolio file

On your new repository page, tap Add file → Upload files.

Choose your downloaded portfolio (the file ending in .html) from your Files app, then tap Commit changes at the bottom.

Important: your file must be named index.html to load automatically. If it isn't, tap the file on GitHub, tap the pencil icon, and rename it to index.html before saving.

Updating your portfolio later

Edited your portfolio and downloaded a new version? Just repeat step 3 — upload the new index.html and commit. The new version replaces the old one within a couple of minutes.

Optional level-up

Want yourname.com instead?

Your free .github.io address works forever. If you'd like a custom domain (around $15–20/year):

  • Buy a domain from any registrar (Namecheap, GoDaddy, VentraIP…)
  • In your repository: Settings → Pages → Custom domain — type your domain
  • At your registrar, add the DNS records GitHub shows you (they give exact values)
  • Tick Enforce HTTPS once it's verified

This step is completely optional and can be done any time later.
