Licence key generator

Create a customer licence key

Type (or generate) the key you'll give your customer, then copy its hash into the portfolio file. The customer only ever sees the key — never the hash.

  1. Type a key (or tap Suggest a key), then Generate hash and Copy hash.
  2. Open the customer's portfolio file and find var LIC_HASHES = [ in the engine script.
  3. Add your copied hash as a new line inside the brackets, in quotes, with a comma after the previous line.
  4. Remove the demo hash line before selling.
  5. Send the customer their key (not the hash). They tap Enter key in the editor toolbar to activate.
